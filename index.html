Git commands:

Version check: git -v


Check User Name: git config --get user.name

Check User Email: git config --get user.email


Change User Email: git config --global user.email

Change User Email: git config --global user.email


Initialized Git repository: git init

Commit any changes to the Stage: git commit -m "(mesgShort) Your message"

Track and commits the changes to local Repo: git add [file.Name]

Check status of the changes: git status

Show the commit history of the repository: git log

Show the commit history in Oneline: git log --oneline

Show the commit history with reset: git reflog


Git reset commit from stage: git reset --hard [id]

Remove commited file from Stage: git rm --cached [filename]

Remove & Delete commited file from Stage & Folder: git rm [filename]

Ignore any file to track in Git: create a file named '.gitignore'
and add the file name in the .gitignore file
