Git commands:

Version check: git -v


User info config

Check User Name: git config --get user.name

Check User Email: git config --get user.email


Change User Email: git config --global user.email

Change User Email: git config --global user.email


Git Initialization Commands

Initialized Git repository: git init

Commit any changes to the Stage: git commit -m "(mesgShort) Your message"

Track and commits the changes to local Repo: git add [file.Name]

Check status of the changes: git status

History check

Show the commit history of the repository: git log

Show the commit history in Oneline: git log --oneline

Show the commit history with reset: git reflog


Restoration command

Git reset commit from stage: git reset --hard [id]

Remove commited file from Stage: git rm --cached [filename]

Remove & Delete commited file from Stage & Folder: git rm [filename]

Ignore any file to track in Git: create a file named '.gitignore'
and add the file name in the .gitignore file


Branching commands

Check the current branch: git branch

Check all branch list: git branch --list

List all branches: git branch -a

List all remote branches: git branch -r

Check the current branch with details: git branch -vv

Create a new branch: git branch [branchName]

Switch to another branch: git switch [branchName]

Switch to another branch and create a new branch: git switch -b [branchName]

Delete a branch: git branch -d [branchName]

Force delete a branch: git branch -D [branchName]

Rename a branch: git branch -m [oldBranchName] [newBranchName]

Merge a branch to current branch: git merge [branchName]

Merge a branch to current branch with no fast forward: git merge --no-ff [branchName]

Merge a branch to current branch with fast forward: git merge --ff [branchName]

Show changes

Show the changes made in the file: git diff [file.Name]
